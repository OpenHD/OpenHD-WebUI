Introduction to OpenHD

This offline copy mirrors the OpenHD documentation page so you can continue reading when openhdfpv.org is unavailable.

OpenHD is a suite of software designed for long-range video transmission, telemetry, and RC control. While originally built with hobbyist drones in mind, it adapts to a wide range of other applications.

Current Version

You are viewing the documentation for the latest development version. Use the version selector on the main documentation site to switch between OpenHD releases.

What OpenHD Can Do

Our software stack can transmit all of the following over a single link:

  • High definition video, including multiple camera streams
  • Two-way UAV telemetry
  • Two-way OpenHD telemetry for settings, range adjustments, channel changes, Wi-Fi modulation, and more
  • RC control signals

All of these signals travel over a single transmission channel, with telemetry using MAVLink for high compatibility and reliability.

How the OpenHD Link Works

OpenHD relies on readily available Wi-Fi adapters, but not in the typical consumer Wi-Fi configuration. To achieve long-range, low-latency performance we configure the adapters to behave similarly to analog video transmitters. The protocol, originally pioneered by befinitiv, has been reimplemented and refined by the OpenHD community.

This configuration enables high quality video and control data to travel great distances while keeping latency low.

QOpenHD App

We provide a multi-platform QOpenHD app with a customizable on-screen display for live video. The app integrates tightly with the rest of the OpenHD suite so you can monitor and manage your link in real time.
